How to use on Netlify:
1. Upload all files to GitHub repository.
2. Connect repository with Netlify.
3. Netlify Dashboard > Site configuration > Identity > Enable Identity.
4. Identity > Services > Enable Git Gateway.
5. Registration preference: Invite only.
6. Invite your email.
7. Open https://your-site.netlify.app/admin and login.

Public website: /
Admin panel: /admin/
Appointment form submissions: Netlify > Forms > appointment
